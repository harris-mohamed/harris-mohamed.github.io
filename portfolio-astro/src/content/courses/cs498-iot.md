## Course Description

CS 498: Internet of Things had several projects that introduced students to networking and general IoT applications. The most prominent and relevant of these was AWS. The course covered the full spectrum of IoT development from device programming to cloud integration.

## Topics Covered

- **IoT device programming** - Arduino and Raspberry Pi development
- **IoT protocol stacks** - Zigbee, 5G, NFC, MQTT, and more
- **Networking backhaul design** - Infrastructure and connectivity
- **Security enforcement** - IoT security best practices
- **Cloud-based IoT platforms** - AWS IoT services and integration
- Device-to-cloud communication
- Edge computing concepts
- IoT system architecture

## Key Projects

- Arduino-based IoT device programming
- Raspberry Pi system integration
- AWS IoT platform implementation
- Protocol stack implementation and comparison
- End-to-end IoT solution development

## Skills Gained

- Arduino programming
- Raspberry Pi development
- IoT protocol implementation (Zigbee, 5G, MQTT, NFC)
- AWS IoT platform expertise
- Network design for IoT
- IoT security practices
- Device-to-cloud integration
- Embedded systems programming
